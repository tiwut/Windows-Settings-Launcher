The **Windows Settings Launcher** is a utility application built with Python and the Tkinter library, designed to provide quick and easy access to a comprehensive list of Windows settings, utilities, and Control Panel applets using their respective command-line executables (.msc, .cpl, .exe, and ms-settings URIs).

‚ú® Key Features

  • Extensive Settings List: Access over 40 different system tools, network settings, security options, and classic Control Panel items.
  • Dark Mode Interface: Features a modern, accessible dark mode theme implemented with Tkinter's ttk.Style.
  • Real-time Search: Easily find the desired setting using a responsive search bar.
  • Quick Launch: Launch settings via double-click, Enter key, or a dedicated "Launch Setting" button.
  • Windows-Native Launch: Uses os.startfile() and subprocess.run() to execute Windows-specific commands efficiently.

üîß How It Works

The application relies on a comprehensive Python dictionary named SETTINGS_COMMANDS that maps user-friendly setting names (e.g., "Device Manager") to their corresponding Windows commands (e.g., devmgmt.msc, ms-settings:display). When a user selects an item and triggers the launch, the associated command is executed by the operating system.

Example Commands:

"Device Manager": "devmgmt.msc",
"Display Settings": "ms-settings:display",
"Programs and Features (Classic)": "appwiz.cpl"
